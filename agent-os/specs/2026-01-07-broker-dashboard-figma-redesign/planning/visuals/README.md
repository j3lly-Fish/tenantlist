## Key Design Elements

### Navigation
- Sidebar width: ~250px
- Menu items with icons
- Three-column layout

### Modals
- Max width: 600px
- Rounded corners: 8px
- Step indicators (e.g., "2 of 2: Basic Information")

### Buttons
- Primary: Dark (#1a1a1a), white text
- Secondary: White, outlined, dark text
- Height: 40px
- Border radius: 8px

### Cards
- White background
- 1px border (#e0e0e0)
- 8px border radius
- 24px padding

### Typography
- SF Pro font family
- Headings: Bold, various sizes
- Body: Regular weight
- Secondary text: #666666

### Colors
- Primary dark: #1a1a1a
- Background: #f5f5f5
- Borders: #e0e0e0
- Accent blue: #4a90e2
- Verified badge: Blue with checkmark
